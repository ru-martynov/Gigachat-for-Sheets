GIGACHAT

Принимает значение (промпт) и возвращает текстовый ответ модели.

GIGACHAT_LIST

Принимает значение (промпт) и возвращает ответ модели в виде списка, заполняя ячейки.

GIGACHAT_TABLE

Принимает значение (промпт) и возвращает ответ модели в виде таблицы с данными на основе запроса.

GIGACHAT_WEB

Принимает URL адрес страницы и запрос (промпт). Возвращает ответ модели на основе данных веб-страницы.
